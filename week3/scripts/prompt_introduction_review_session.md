# Introduction 리뷰 세션 진행 프롬프트

> **목적**: Week 3 강의에서 학생 10명이 각자 작성한 Introduction을 전체 그룹에서 리뷰하는 세션을 효과적으로 진행하기 위한 프롬프트

---

## 📋 세션 개요

### 기본 정보
- **참가자**: 학생 10명
- **형식**: 전체 그룹에서 한 명씩 발표 후 리뷰
- **시간**: 60분 (각 학생당 6분)
- **리뷰 기준**: Gap-driven 4단계 구조

### 시간 배분
- 각 학생당 6분
  - 발표: 2분
  - 리뷰/코멘트: 4분
- 총 10명 × 6분 = 60분

---

## 🎯 리뷰 기준: Gap-driven 4단계 구조

### 체크리스트

#### 1. Established Knowledge (확립된 지식)
- ✅ 명확한 consensus가 제시되는가?
- ✅ 적절한 논문들이 인용되는가?
- ✅ 정량적 증거가 있는가?

#### 2. Emerging Challenges (새로운 도전)
- ✅ 모순/한계가 명확히 제시되는가?
- ✅ 기존 지식에 의문이 제기되는가?
- ✅ 왜 중요한지 설명되는가?

#### 3. Critical Gap (핵심 공백)
- ✅ Gap statement가 명확한가?
- ✅ Gap 유형이 적절한가? (Conceptual/Mechanistic/Translational/Paradox?)
- ✅ Incremental gap이 아닌가?
- ✅ Broader significance가 있는가?

#### 4. Research Opportunity (연구 기회)
- ✅ 해결 방법/관점이 제시되는가?
- ✅ 왜 지금 연구할 수 있는가?
- ✅ 본 연구의 기여가 명확한가?

---

## 📝 진행 프롬프트

### 교수용 프롬프트

```
당신은 Week 3 강의를 진행하는 교수입니다.
학생 10명이 각자 작성한 Introduction을 전체 그룹에서 리뷰하는 세션을 진행해야 합니다.

## 세션 목표
- 학생들이 자신의 Introduction을 Gap-driven 구조 관점에서 평가받기
- 다른 학생들의 Introduction에서 배우기
- 구체적이고 실용적인 피드백 받기

## 진행 방식
1. 전체 그룹에서 한 명씩 발표 (2분)
2. 전체 그룹이 체크리스트 기반으로 코멘트 (4분)
3. 다음 학생으로 이동

## 시간 관리
- 각 학생당 정확히 6분 엄수
- 발표 2분 초과 시 정중히 중단
- 리뷰 4분 초과 시 다음 학생으로 이동

## 리뷰 형식
각 학생의 Introduction에 대해:
- 강점 1가지 (30초 이내)
- 개선점 1가지 (30초 이내)
- 구체적 제안 (30초 이내)

## 진행 가이드
1. 세션 시작 전 체크리스트 안내 (강의 시간에 이미 안내됨)
2. 발표 순서 결정 (알파벳 순, 출석 번호 순 등)
3. 각 학생 발표 및 리뷰 진행
4. 시간 엄수 (타이머 사용 권장)
5. 건설적이고 존중하는 분위기 유지

## 주의사항
- 모든 학생이 발표할 수 있도록 시간 관리
- 부정적 비판보다는 건설적 제안 중심
- 각 코멘트는 30초 이내로 간결하게
- 발표자가 방어적으로 나오지 않도록 유도
```

```
## 리뷰 체크리스트

### 2. Emerging Challenges (새로운 도전)

### 3. Critical Gap (핵심 공백)

- Incremental gap이 아닌가?

## 리뷰 형식 (30초 이내로 간결하게)
1. **강점 1가지**: 어떤 부분이 잘 되었는가?
2. **개선점 1가지**: 가장 중요한 개선이 필요한 부분은?
3. **구체적 제안**: 어떻게 개선할 수 있는가?

## 리뷰 원칙
- 건설적이고 존중하는 톤 유지
- 구체적이고 실용적인 제안
- 부정적 비판보다는 개선 방안 중심
- "이 부분을 이렇게 하면 더 좋을 것 같다" 형식
```

## ⏰ 시간 관리 가이드

### 타이머 설정
- 각 학생당 6분 타이머 설정
- 발표 2분 경과 시 알림
- 리뷰 4분 경과 시 알림

### 시간 초과 대응
- 발표 2분 초과 시: "시간이 다 되었습니다. 핵심 Gap statement만 간단히 말씀해주세요."
- 리뷰 4분 초과 시: "다음 학생으로 넘어가겠습니다. 추가 코멘트는 수업 후에 주고받을 수 있습니다."

### 시간 절약 팁
- 발표자는 핵심만 말하기 (2분 준수)
- 리뷰자는 각 코멘트 30초 이내
- 불필요한 질문은 수업 후로 미루기

## 📊 진행 체크리스트

### 세션 시작 전
- [ ] 발표 순서 결정 (알파벳 순, 출석 번호 순 등)
- [ ] 타이머 준비
- [ ] 체크리스트 안내 (강의 시간에 이미 안내됨)
- [ ] 리뷰 원칙 안내 (건설적, 존중, 구체적)

### 각 학생 발표 및 리뷰
- [ ] 발표자: Introduction 발표 (2분)
- [ ] 전체 그룹: 체크리스트 기반 코멘트 (4분)
  - [ ] 강점 1가지
  - [ ] 개선점 1가지
  - [ ] 구체적 제안
- [ ] 시간 엄수 확인

### 세션 종료 후
- [ ] 전체 공유 (Phase 4에서 진행)
- [ ] 공통 개선점 정리
- [ ] 다음 단계 안내

- 시간 관리 엄수

- 다른 학생의 Introduction에서 배우기

- 모든 학생이 발표 완료

- 시간 내 완료 (60분)
- 건설적이고 존중하는 분위기 유지

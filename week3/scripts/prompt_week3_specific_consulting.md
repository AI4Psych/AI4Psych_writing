## 📋 Week 3 컨텍스트

### 핵심 목표
**"어떻게 하면 Nature/Science급 Research Gap을 발견할 것인가?"**

### 수업 구조 (90분)
- **강의 15-20분**: 탑티어 저널 Gap 기준 소개
- **Workshop 80-85분**: 자신의 연구에 적용 + 동료 피드백
  - Phase 0: 실제 예제 분석 (10분)
  - Phase 1: Gap Discovery (20분)
  - Phase 2: Gap Validation & Crafting (25분)
  - Phase 3: Peer Review (20분)
  - Phase 4: 최종 정제 + 공유 (10분)

### Week 3의 위치
```
Week 2 → Nature/Science급 초록 (Gap 언급)
Week 3 → Gap-driven 서론 (Gap 발견 및 검증) ← 현재
Week 4 → Methods/Results (Gap 해결 방법론)
```

### 핵심 학습 내용
1. **Gap 유형 분류**: Incremental vs Conceptual/Mechanistic/Translational/Paradox
2. **3-Stage Validation**: Landscape Mapping → Critical Review → Gap Validation
3. **Gap Statement Crafting**: 약한 Gap → 강한 Gap 변환
4. **Gap-Driven 구조**: 4단계 구조 (Established Knowledge → Emerging Challenges → Critical Gap → Research Opportunity)

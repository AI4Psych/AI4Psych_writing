## 🔍 Gap-Driven 문헌 리뷰 구조 (상세)

### 단계 1: Established Knowledge (확립된 지식)
> **가이드**: 
> - 여러 연구를 통합해서 "우리가 아는 것"을 제시
> - Consensus, 통합된 발견
> - 적절한 논문 인용
> - 정량적 증거 포함

**체크리스트:**
- [ ] 명확한 consensus가 제시되는가?

> - 왜 중요한지 설명

**체크리스트:**
- [ ] 모순/한계가 명확히 제시되는가?
- [ ] 기존 지식에 의문이 제기되는가?

### 단계 3: Critical Gap (핵심 공백)
> **가이드**:
> - 명확한 Gap Statement 작성
> - Gap 유형 확인: Conceptual/Mechanistic/Translational/Paradox?
> - Incremental gap이 아닌지 확인
> - Broader significance 포함

**Gap 유형 확인:**
- [ ] Conceptual Gap: 기존 이론으로 설명할 수 없는 현상
- [ ] Mechanistic Gap: 작동 원리 불명확
- [ ] Translational Gap: 기초→응용 연결 안 됨
- [ ] Paradox Gap: 모순 결과들 존재
- [ ] ❌ Incremental Gap 아님 (탑티어 부적합)

**체크리스트:**
- [ ] Gap statement가 명확한가?
- [ ] Gap 유형이 적절한가?
- [ ] Incremental gap이 아닌가?
- [ ] Broader significance가 있는가?

**여기에 작성하세요:**

## ✅ 최종 점검 체크리스트

### 깔때기 구조 (전체 흐름)
- [ ] 우리가 모르는 것 → 중요한 질문 → 나의 연구 질문 → 우리의 어프로치
- [ ] 각 단계가 명확히 구분되는가?
- [ ] 논리적 흐름이 자연스러운가?

### Gap-driven 4단계 (문헌 리뷰 부분)
- [ ] Established Knowledge → Emerging Challenges → Critical Gap → Research Opportunity
- [ ] 각 단계가 명확히 제시되는가?

### 두 구조의 연결
- [ ] "우리가 모르는 것"이 Critical Gap과 연결되는가?
- [ ] "중요한 질문"이 Emerging Challenges에서 도출되는가?
- [ ] "나의 연구 질문"이 Critical Gap의 구체화인가?
- [ ] "우리의 어프로치"가 Research Opportunity와 연결되는가?

### Nature Editor 관점 평가
- [ ] 첫 2-3 문단에서 novelty가 명확한가?
- [ ] 논리적 흐름이 명확한가?
- [ ] 진짜 gap인가, incremental gap인가?
- [ ] 각 단계가 명확히 구분되는가?

## 📝 AI 활용 가이드

### 프롬프트 레시피 1: 깔때기 구조 분석
```
다음 Introduction을 깔때기 구조로 분석해줘:

[Introduction 텍스트]

다음을 확인해주세요:

1. 우리가 모르는 것 (What we don't know)
   - 넓은 지식 공백이 명확히 제시되는가?
   - Critical Gap과 연결되는가?

2. 중요한 질문 (Important question)
   - 그 공백이 왜 중요한가가 설명되는가?
   - Emerging Challenges와 연결되는가?

3. 나의 연구 질문 (My research question)
   - 구체적이고 연구 가능한 질문으로 좁혀지는가?
   - Critical Gap의 구체화인가?

4. 우리의 어프로치 (Our approach)
   - 해결 방법이 제시되는가?
   - Research Opportunity와 연결되는가?

각 단계가 명확히 구분되어 있는지, 논리적 흐름이 자연스러운지 평가해주세요.
```

### 프롬프트 레시피 2: Gap-driven 4단계 평가
```
다음 Introduction을 Gap-driven 4단계 관점에서 평가해줘:

[Introduction 텍스트]

평가 기준:

1. Established Knowledge
   - 명확한 consensus가 제시되는가?
   - 적절한 논문들이 인용되는가?
   - 정량적 증거가 있는가?

2. Emerging Challenges
   - 모순/한계가 명확히 제시되는가?
   - 기존 지식에 의문이 제기되는가?
   - 왜 중요한지 설명되는가?

3. Critical Gap
   - Gap statement가 명확한가?
   - Gap 유형이 적절한가? (Conceptual/Mechanistic/Translational/Paradox?)
   - Incremental gap이 아닌가?
   - Broader significance가 있는가?

4. Research Opportunity
   - 해결 방법/관점이 제시되는가?
   - 왜 지금 연구할 수 있는가?
   - 본 연구의 기여가 명확한가?

개선 제안을 구체적으로 제시해주세요.
```

### 프롬프트 레시피 3: Gap Statement 강화
```
다음 gap statement를 Nature/Science 수준으로 강화해줘:

[초안]

다음 요소들을 포함해서:
1. 왜 이 gap이 중요한가 (broader significance)
2. 기존 연구들이 이 gap을 해결하지 못한 이유
3. 이 gap을 해결하면 얻을 수 있는 conceptual/practical advance
4. 3가지 버전으로 제시 (각각 다른 framing 전략)
```
